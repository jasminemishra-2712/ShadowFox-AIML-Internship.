# ShadowFox AIML Internship Projects

This repository contains all the projects completed as part of the **ShadowFox Artificial Intelligence and Machine Learning Internship Program**. The tasks include Beginner, Intermediate, and Advanced level projects covering Machine Learning and Natural Language Processing concepts.

---

# 📌 Beginner Level Project

## Boston House Price Prediction
**Objective:**  
To predict housing prices using Machine Learning regression techniques.

**Description:**  
In this project, the Boston Housing dataset was used to train a Linear Regression model. The dataset was first loaded using Pandas, and missing values were handled using mean imputation. The dataset was then divided into training and testing sets. The Linear Regression algorithm was applied to predict housing prices based on different features such as crime rate, number of rooms, tax rate, etc.

**Technologies Used:**
- Python
- Pandas
- NumPy
- Scikit-learn

**Result:**  
The model successfully predicted house prices and achieved a good R2 Score, showing effective regression performance.

# 📌 Intermediate Level Projects

## Car Price Prediction
**Objective:**  
To predict the selling price of cars using Machine Learning.

**Description:**  
This project uses a car dataset containing information such as fuel type, transmission, present price, and kilometers driven. Data preprocessing was performed by converting categorical data into numerical format. A Random Forest Regression model was trained to predict the car selling price.

**Technologies Used:**
- Python
- Pandas
- Scikit-learn
- Random Forest Regression

**Result:**  
The model achieved high accuracy in predicting car prices.

---

## Loan Approval Prediction
**Objective:**  
To predict whether a loan application will be approved or rejected.

**Description:**  
The loan dataset contains applicant information such as income, education, credit history, and loan amount. After preprocessing and encoding categorical variables, a Random Forest Classification model was trained to predict loan approval status.

**Technologies Used:**
- Python
- Pandas
- Scikit-learn
- Random Forest Classifier

**Result:**  
The model successfully predicted loan approval status with good accuracy.

# 📌 Advanced Level Project

## NLP Language Model using GPT-2
**Objective:**  
To implement a Natural Language Processing (NLP) model capable of generating human-like text.

**Description:**  
In this project, the HuggingFace Transformers library was used to load the pretrained GPT-2 language model. The model was used to generate meaningful text based on different input prompts. The generated outputs were analyzed and visualized to understand the performance of the language model.

**Technologies Used:**
- Python
- HuggingFace Transformers
- PyTorch
- Jupyter Notebook
- Matplotlib

**Result:**  
The model successfully generated coherent and context-based text, demonstrating the power of modern NLP models.
